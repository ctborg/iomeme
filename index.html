    • iomeme

    A fast and easy way to create the Internet's most popular memes

    Project maintained by justinjpacheco Hosted on GitHub Pages — Theme by mattgraham

    About

    iome.me is a web service used to quickly create the Internet's most popular memes

    Examples

    • http://iome.me/ss
    • http://iome.me/ss/hello
    • http://iome.me/ss/hello/world

    Usage

    Every iome.me request takes three parameters:

    /type/top/bottom

    Where:

    • type (the type of meme) required
    • top (top text) optional
    • bottom (bottom text) optional

    Inserting spaces

    You can use a "+" to insert a space into your text

    • http://iome.me/ss/hello+world

    Or you can use the url escaped value (%20)

    • http://iome.me/ss/hello%20world

    Uploading to Imgur

    When uploading images to Imgur, you can add "/.jpg" to the end of your iome.me url like so

    • http://iome.me/sk/uploads+to+imgur/it+works/.jpg

    We have to do this because Imgur expects the file extension to be at the end of the URL.
